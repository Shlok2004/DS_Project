Getting Started

First install next.js in root of project directory:
npm install

Then create a virtual python environment using:
python -m venv venv

Activate the python virtual server with:
venv\Scripts\activate

Then you can start the frontend with:
npm run dev

Open http://localhost:3000 with your browser to see the result.

Then we can setup the backend by first installing all required files in the following order:
pip install torch==2.2.0
pip install -r requirements.txt

To start the FastAPI server, navigate to the backend folder and run:
uvicorn main:app --reload

Supabase Setup:
Create a table with the following schema in PostgreSQL:

create table public.call_severities (
  id bigint generated by default as identity not null,
  created_at timestamp with time zone not null default now(),
  emotional_sev real null default '0'::real,
  context_sev real null default '0'::real,
  transcript text null default ''''''::text,
  key_details json null default '{}'::json,
  is_active boolean null default true,
  emotions text null default ''::text,
  constraint call_severities_pkey primary key (id)
) TABLESPACE pg_default;

API KEYS:
In the project root, you will need to create a .env file in the following format:
MISTRAL_API_KEY = <YourKeyHere>
SUPABASE_URL=postgresql://postgres:<YourKeyHere>
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000
